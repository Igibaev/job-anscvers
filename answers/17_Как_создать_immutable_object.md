
Для создания своего неизменяемого (immutable) класса в Java необходимо выполнить следующие шаги:

Сделать класс final, чтобы его нельзя было наследовать.
Сделать все поля private final, чтобы они не могли быть изменены после создания объекта.
Создать конструктор, который принимает все поля в качестве аргументов и устанавливает их значения с помощью this.
Создать геттеры для всех полей, но не создавать сеттеры, чтобы не допустить изменения значений полей после создания объекта.
Пример неизменяемого класса Person:

```java
public final class Person {
private final String name;
private final int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }
}
```

Такой класс будет неизменяемым, поскольку его состояние определяется только при создании объекта, и далее оно не может быть изменено. Неизменяемые классы в Java имеют ряд преимуществ, например, они безопасны в многопоточной среде и могут использоваться в качестве ключей в хеш-таблицах.
