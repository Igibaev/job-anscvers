В Java ключевое слово final может использоваться для обозначения неизменяемых сущностей, таких как переменные, методы и классы.

Для переменных: значение переменной не может быть изменено после инициализации.
Для методов: метод не может быть переопределен в подклассах.
Для классов: класс не может быть наследован.
Также ключевое слово final может использоваться для указания констант, которые не могут быть изменены.

Использование final может повысить производительность и безопасность кода, так как компилятор может проводить оптимизации при работе с неизменяемыми сущностями, а также гарантируется, что значение не будет изменено случайно или намеренно.
