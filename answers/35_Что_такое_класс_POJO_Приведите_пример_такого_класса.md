POJO (Plain Old Java Object) - это класс Java, который следует простым стандартам JavaBeans и не зависит от какой-либо специальной библиотеки или фреймворка.

Пример класса POJO:

```java 
public class User {
private String name;
private int age;

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // геттеры и сеттеры для полей name и age

    public String toString() {
        return "User(name=" + name + ", age=" + age + ")";
    }
}
```
Это простой класс Java, который представляет пользователя с именем и возрастом, содержит конструктор для их инициализации и методы доступа для получения и установки значений полей. Также он переопределяет метод toString(), чтобы вернуть строковое представление объекта. Этот класс можно использовать в различных приложениях, где требуется представление пользователя.
